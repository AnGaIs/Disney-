## :computer: Instalação

Primeiro, certifique-se de ter o Expo CLI instalado: `npm install -g expo-cli`

Use: `yarn` or `yarn install`

Rode o projeto: `expo start` or `yarn dev`

## :blue_book: Informações

- Expo SDK 39
- iOS and Android
- React Navigation v4
- PropTypes
